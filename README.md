# web-scrapping-sbrbank
Учебный проект, демонстрирующий:
- Посещение сайта Центробанка России `requests.get(url)` (https://www.cbr.ru/),
- Поиск на главной странице блока с курсами валют, извлечение курсов доллара (USD), евро (EUR) и китайского юаня (CNY) с помощью парсера (`.find_all`, `.find`, `BeautifulSoup`),
- Вывод их на экран в красивом виде (`print`),
- Сохранение результат в JSON-файл в папку "data" (`Path`, `.parent /`, `json.dump`)
- Пути обхода блокировок (`headers`, `.sleep`)
